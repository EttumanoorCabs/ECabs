# EttumanoorCabs

EttumanoorCabs is a web application for a tour cab service. It showcases a fleet of cars available for rent (with driver), provides detailed information about each car, and allows users to contact the service for bookings.

## Features

- Display a list of available cars with images and short descriptions.
- Detailed modal view for each car with additional information and features.
- Responsive design for various screen sizes and aspect ratios.
- Contact section with phone, email, and location information.
- Smooth scrolling and interactive UI elements.

## Technologies Used

- HTML
- CSS
- JavaScript
